-  ✔ quantidade horas
-  ✔ valor hora
-  ✔ recebe auxilio educação (SIM => +400, Não +0)
-  ✔ vale onibus (SIM => -6%, Não -0%)
-  ✔ vale creche (SIM => +200, Não 0)

-  Imposto de Renda:

	- De R$ 3.751,06 até R$ 4.664,68: 22,5%
	- Acima de R$ 4.664,68: 27,5%

	-  De R$ 3.856,95 até R$ 7.507,49   14 %
